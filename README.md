## 🔐 Sistema de Login

Projeto simples de login desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Validação de usuário e senha
- Interface simples e centralizada
- Feedback de login (sucesso/erro)

## 🛠 Tecnologias
- HTML
- CSS
- JavaScript

## 📚 Aprendizados
- Manipulação do DOM
- Captura de dados com input
- Estrutura de funções em JavaScript
